 커뮤니티 서버는 그룹 서버에서 다루지 않는 기능을 제공할 때 사용합니다.

 여기서 그룹 서버가 하지 않는 업무는 다음을 뜻합니다.

1. 서버 내 콘텐츠 편집처리, 기기 변경 이메일 발송, 이용자 수 집계 등과 같이 Nakama 기본 설정 외 행동을 서버에서 진행해야하는 경우.

2. 만능 스캔 도구 등 Nakama 기능을 사용하나 편의 등을 위한 중계 서버 용도.

3. 비회원이 사용가능한 기능/도구의 서버 역할.

 커뮤니티 도구 및 서비스로 제공되나 사용자 구분이 필요한 경우 그룹 서버의 회원정보를 이용합니다.

 커뮤니티 서버는 그룹 서버와 같은 컴퓨터에서 병행 운영중으로 개발자 판단에 의해 언제든 운영중단되거나 데이터 초기화를 진행할 수 있습니다. 커뮤니티 서버를 사용하는 것은 서버에서 이용한 활동정보가 고지없이 손실될 수 있다는 것에 동의한 것으로 간주됩니다.
