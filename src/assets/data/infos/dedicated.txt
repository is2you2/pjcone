 이 앱을 사용하는 모바일 기기로 사설 서버를 개설할 수 있습니다.

 아래 준비된 모든 서버는 보안처리가 되어있지 않습니다.
 신뢰할 수 없는 네트워크에서 사용할 경우 그림자 속에서 일하는 누군가가 통신 데이터를 가로채 읽거나 변형할 수 있습니다.

 농담같지만 중요한 이야기니까 꼭 기억해주세요 :)

 네트워크를 전혀 이용할 수 없는 상황이라면 모바일 핫스팟을 네트워크 삼아 핫스팟에 연결된 기기들과 통신할 수 있습니다.
 
-----------------------
기능 서버에 대해서

 사용하는 컴퓨터와 같은 공유기를 사용하고 있다면 웹 페이지에서 제공하는 일부 기능들을 보다 편리하게 사용할 수 있도록 일종의 리모콘 기능을 제공하고 있습니다. 이 글은 아래 기능 서버들에 해당하는 공통사항입니다.
 
 각 기능마다 구성이 다르기 때문에 별도의 서버구조를 가지고 있고 서버와 웹 사이트가 1:1로만 접속할 수 있습니다. 이 후 새로운 연결에 대해서는 무시합니다.
 기능 서버들은 작업 편의와 보안 이슈를 최소화하기 위해 웹 사이트와의 연결이 끊어지면 자동으로 서버가 종료되도록 구성되어 있습니다.
