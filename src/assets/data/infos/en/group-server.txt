* Virtually open information notice
-----------

 Nakama Server is an open source server, anyone can create and manage a private server if they wish.
 As the name suggests, the development test server is operated for the purpose of development testing, so it may be suspended or data initialized at any time by the developer's judgment. By using the development test server to use the features provided by this app, you agree that user information and activity information stored on the server may be lost without notice.

* Virtually open information notice
 Nakama is an open source server, so anyone can configure a private server and implement additional features on the server as needed.
 In other words, the manager of the private server can access/store/modify/delete all information communicating with the server for any purpose, and can add new functions.
 This developer does not take any responsibility for any problems caused by using an unreliable server. We strongly recommend that you use servers provided by reliable institutions, companies, groups, etc.
