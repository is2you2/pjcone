 커뮤니티 서버는 그룹 서버가 하지 않는 업무를 처리할 때 사용합니다.

 여기서 그룹 서버가 하지 않는 업무는 다음을 뜻합니다.

1. 서버 내 콘텐츠 편집처리, 기기 변경 이메일 발송, 이용자 수 집계 등과 같이 Nakama 기본 환경 외 행동을 서버에서 진행해야하는 경우

2. Nakama를 사용하지 않은 구버전 앱들의 서버 역할.

3. 비회원이 사용가능한 기능/도구의 서버 역할.

 커뮤니티 서버에서 제공하는 회원 서비스 이용시 회원정보는 개발 테스트 서버의 회원정보를 이용하며 비회원 활동시에는 회원 식별 정보를 기록하지 않습니다.
 커뮤니티 서버와의 통신은 보안처리가 되어있습니다. (LetsEncrypt를 이용한 SSL 구축)

 커뮤니티 서버는 개발자 개인 컴퓨터에서 운영중으로 개발자 개인의 상황이나 판단에 의해 언제든 운영중단되거나 데이터 초기화를 진행할 수 있습니다.
 이 앱에서 서비스되는 기능을 통해 커뮤니티 서버를 이용한다면 이러한 불편함에 동의하는 것으로 간주합니다.
