 커뮤니티 서버는 그룹 서버가 하지 않는 업무를 처리할 때 사용합니다.

 여기서 그룹 서버가 하지 않는 업무는 다음을 뜻합니다.

1. 서버 내 콘텐츠 편집처리, 기기 변경 이메일 발송, 다중기기 이용 허락 등과 같이 Nakama 서버의 본래 목적과 다른 행동을 서버에서 진행해야하는 경우

2. Nakama를 사용하지 않은 구버전 앱들의 서버 역할.
예) EngineWorksPPT 등

3. 비회원이 사용가능한 기능/도구의 서버 역할.
예) 랜덤채팅 등

 이 서버는 그룹서버와는 달리 개발자가 직접 개발/구축한 서버로, 사용자 사설서버로 사용할 수 없습니다.
 다만, DIY 사용자 도구가 서버를 필요로 한다면 해당 앱과 통신할 별도 서버를 직접 구축한 후 사용할 수 있습니다.
